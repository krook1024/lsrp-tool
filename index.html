Introduction

This this tool written in AutoHotKey provides you with useful keybinds when playing LS:RP, it's designed mostly for civilians but everyone can feel free to use them. Also, there is a basic friend system included where you can add, remove and check the online friends of yours.

Installation

It's simple, you can either download the executable or run the source itself if you have AutoHotKey installed on your PC.

Usage

You just run the executable or the file called LS-RP.ahk in order to get the hotkeys working. They only work while in-game.

Hotkeys and commands

Hotkeys

  • ALT+1 — Enter/exit a property/business what so ever.
  • ALT+2 — Show your custom designed /ame from the Settings.ini file.
  • ALT+3 — Open/close a door in a house.
  • ALT+K — Start/stop the engine of a vehicle.
  • ALT+L — Use the /lock command to lock/open a house, vehicle, property, whatever.
  • ALT+M — Put on/remove an OOC mask.
  • Numpad 5 — View general information (health;armor) about your character.
  • Numpad 7 — Put on/remove your custom clothing, slots are set in Settings.ini.
  • Numpad + — Open a toll.

Commands

  • /cp — Shorter version of /clothing place.
  • /lk — Does the same as ALT+2.

Friends system

  • /fadd — Add an online friend by ID/name.
  • /fdel — Remove an online friend from the list.
  • Numpad 4 — Show a list of the online friends.

Settings.ini

  • Ame= — Place your custom designed /ame here.
  • InfoStyle= — Gametext/Chatbox
  • Clothes= — Your active clothing items, seperated with commas (,).
